# 交付前排版检查表

## 模式与页面

- [ ] 已明确使用毕业论文模式或期刊模式，未混用两套结构。
- [ ] 纸张、页边距、装订线、分节和单双栏正确。
- [ ] 封面/前置部分/正文的页码起算和格式正确。
- [ ] 没有用连续空行、空格或手工换行模拟分页、缩进或对齐。

## 字体与段落

- [ ] 中文、英文/数字分别使用规定字体。
- [ ] 正文、摘要、标题、题注、参考文献字号统一。
- [ ] 正文两端对齐、首行2字符、行距和段距一致。
- [ ] 变量斜体、单位正体、上下标和希腊字母规范。
- [ ] 数值与单位不在行尾拆开。

## 标题与目录

- [ ] 标题层级和编号连续，未手工输入自动编号。
- [ ] 标题没有落在页/栏末行，且与后文保持同页/同栏。
- [ ] 毕业论文目录由标题字段生成，层级、点引导线和页码正确。
- [ ] 期刊模式没有误加目录或“第N章”。

## 图表与公式

- [ ] 图表均在首次引用附近，顺序和编号连续。
- [ ] 图片未变形，分辨率足够，坐标、图例和单位可读。
- [ ] 毕业论文采用“图/Figure”；期刊采用“图/Fig.”。
- [ ] 图题在图下，表题在表上，中英文题注顺序正确。
- [ ] 表格不超出版心/栏宽，表头跨页重复，单位放在表头。
- [ ] 公式居中、编号右对齐，编号方式与模式一致。
- [ ] 所有图、表、公式的正文交叉引用均能更新。

## 中英文与内容一致性

- [ ] 中英文题名、摘要、关键词中的技术含义和数值一致。
- [ ] 缩写首次出现时给出全称，后续写法一致。
- [ ] 中文/英文标点、空格、大小写和专名拼写正确。
- [ ] 未因排版修改公式、符号、数据或引用含义。

## 引文与参考文献

- [ ] 文内引用均有文末条目，文末条目均在正文出现。
- [ ] 顺序编码连续，新增/删除文献后已重新排序。
- [ ] 作者、题名、文献类型、期刊/出版社、年份、卷期、页码、DOI已核对。
- [ ] 悬挂缩进、字体、字号、行距和标点统一。
- [ ] 期刊中文文献的英文翻译与中文条目一一对应。
- [ ] 没有猜测缺失的DOI、英文题名、作者拼音或出版信息。

## 最终更新与视觉检查

- [ ] 已更新全部域、目录、页码、题注和交叉引用。
- [ ] 已检查孤行、寡行、异常空白页、重叠、裁切和跨栏顺序。
- [ ] 已检查至少：首页/摘要页、目录页、每章首页、含图表页、参考文献首页和末页。
- [ ] DOCX可编辑且无损坏；如需PDF，已对照DOCX逐页检查PDF导出结果。
